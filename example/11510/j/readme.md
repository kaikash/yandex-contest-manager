J. Прибавить единицу
====================

Ограничение времени

1 секунда

Ограничение памяти

64Mb

Ввод

стандартный ввод или input.txt

Вывод

стандартный вывод или output.txt

Напишите программу, которая прибавляет 1 к натуральному числу N (длина числа N до 1000 знаков)

Формат ввода
------------

Дано чиcло N.

Формат вывода
-------------

Выведите число N+1.

Ввод

Вывод

1

2

### Пример 2

Ввод

Вывод

5

6

Вывод
